Nguyễn Hoàng Tiến - Portfolio Lập Trình Viên IT

1. Mục tiêu nghề nghiệp

Là một sinh viên năm cuối chuyên ngành Công nghệ Thông tin, tôi đang tìm kiếm cơ hội vị trí Thực tập sinh IT để áp dụng tư duy logic và kiến thức phát triển phần mềm vào môi trường doanh nghiệp thực tế.

2. Kỹ năng chuyên môn

  • Ngôn ngữ lập trình: Tư duy xử lý logic tốt với Java, Python và C#.
  • Cơ sở dữ liệu: Thiết kế và truy vấn thành thạo với SQL Server.
  • Ngoại ngữ: Khả năng đọc hiểu tài liệu kỹ thuật (Tiếng Anh B1).

3. Dự án tiêu biểu

  • Hệ thống Quản lý Nhà hàng: Thiết kế Use Case, giao diện và lập trình chức năng quản lý.
  • Hệ thống An ninh Nông trại (IoT): Ứng dụng giao thức MQTT và Node-RED xử lý dữ liệu cảm biến từ ESP32.
  • Game 2D Unity: Lập trình hệ thống logic và UI bằng C#.
